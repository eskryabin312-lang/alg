# SimplePaint
## Приложение для быстрого и удобного рисвания
Эта программа была создана для тех, кто хочет творить и созидать, но не хочет разбираться в сложном управлении других программ.
В SimplePaint присутствуют такие функции как:
- Смена цветов и размеров кисти по нажатию клавиш
- палитра цветов
- управление кистью на стрелочки
- кастомизация внешнего вида кисти(круг, треугольник, черепашка)
- перемещение кисти на ЛКМ
